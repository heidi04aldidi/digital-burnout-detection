# Digital Burnout Detection

## Overview
This project focuses on detecting digital burnout using behavioral, lifestyle, and self-reported indicators.  
It follows a **human-centered data science approach**, aiming to raise awareness rather than diagnose mental health conditions.

## Problem Statement
With increasing screen time, remote work, and digital overload, burnout has become common but under-detected.  
This project builds a data-driven system to identify burnout risk levels.

## Features Considered
- Daily screen time
- Sleep duration
- Work hours
- Stress level
- Physical activity
- Mood rating

## Tech Stack
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib / Seaborn
- Jupyter Notebook
